Tutorial

Updated workflow for Glyph Composer + Audio Converter.

1. Quick Start

  1. Open Glyph Composer.
  2. Create frames and tune frame durations in the timeline.
  3. Load your audio .ogg file (or use the converter flow).
  4. Click Bake & Download OGG.

2. Drawing Tools

Draw / Erase

Paint directly on the 13x13 matrix with drag support.

Fill

Flood-fill connected LEDs with current brightness.

Line

Click start and end to draw straight lines quickly.

Mirror X / Y

Mirror edits for symmetric animations.

Onion

Shows neighboring frame context while editing motion.

Pick

Reads average lit brightness from current frame.

3. Timeline + Audio Sync

  • Timeline mode: frame-by-frame editing with per-frame duration slider.
  • Audio Sync mode: frame blocks aligned along audio time with playback/seek.
  • Resize frame duration directly from frame edges in Audio Sync mode.

Tip: If animation total is longer than audio, export trims to audio length to keep ringtone timing stable.

4. Audio Converter Page

  • Upload MP3/M4A/WAV and preview waveform immediately.
  • Edit before convert: Trim start/end, append silence, and volume.
  • Convert to .ogg (Vorbis), then use Use in Glyph Composer.
  • Info buttons (i) explain each edit option inline.

5. Import / Export

  • Load JSON / Export JSON for animation exchange.
  • Load .glyphproj / Export .glyphproj for full project state.
  • OGG bake writes required glyph metadata tags automatically.

6. Gallery

  • Published animations come from gallery/gallery-index.json.
  • Use Play to preview, Use to load into composer.
