Tutorial

Complete guide for the current Glyph Composer workflow.

1. Quick Start

  1. Open Glyph Composer.
  2. Create frames (or load an existing JSON/project).
  3. Use stack actions in the timeline to control duration and structure.
  4. Select an .ogg file and click Bake & Download OGG.

2. Drawing Tools

Draw / Erase

Paint directly on the 13x13 matrix. Drag works for continuous editing.

Fill

Flood fill on connected LEDs using the current brightness.

Line

Click start point, then end point to draw a line.

Mirror X / Y

Mirror edits across horizontal and/or vertical axis.

Onion

Shows previous/next frame overlay to improve motion editing.

Pick

Reads average lit brightness from current frame and sets brush value.

3. Timeline (Stacks)

The timeline is stack-based. Each stack is a block of repeated frames and can be expanded for fine edits.

Stack Actions

  • Duration: open modal and set stack length in milliseconds.
  • Expand / Hide: show individual frames for detailed editing.
  • Split: divide a stack into two stacks.
  • Merge: merge with the next stack.
  • Dup: duplicate the stack.
  • Delete: remove the stack.

Batch Actions

  • Select multiple stacks using checkboxes.
  • Use Select All, Delete Selected, Duplicate Selected.

4. Import / Export

Animation JSON

  • Load JSON imports animation frames.
  • Export JSON exports current animation in packed frame format.

Project Format (.glyphproj)

  • Export .glyphproj saves full editor state (stacks, groups, current frame, brightness, title).
  • Load .glyphproj restores the full project for editing.

Autosave

  • The editor autosaves to local browser storage.
  • On reload, you can restore the previous session.

5. OGG Workflow (Recommended)

  1. Open your MP3 in Audacity.
  2. Press Ctrl + A.
  3. Go to File -> Export Audio.
  4. Format: Opus file.
  5. Leave the other settings unchanged.
  6. Export.
  7. Rename .opus to .ogg.
  8. Use this .ogg in Glyph Composer export.

Why this matters: the composer writes glyph data into OGG metadata. A clean OGG/Opus source avoids compatibility issues.

6. Gallery Workflow

  • Open Gallery to preview published JSON animations.
  • Click Play to preview, Use to load into the composer.
  • Published entries are controlled via gallery/gallery-index.json.

7. Professional Editing Tips

  • Block out motion first with 3-8 key stacks, then refine inside expanded stacks.
  • Use lower brightness for transitions and higher brightness for impact frames.
  • Use Undo / Redo aggressively when trying alternatives.
  • Save milestone versions as .glyphproj before major changes.
