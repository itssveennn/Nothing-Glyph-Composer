Tutorial for .ogg

Follow these steps to prepare a ringtone file so Glyph export works correctly.

How To Create A Compatible OGG File

  1. Open your MP3 ringtone file in Audacity.
  2. Press Ctrl + A to select everything.
  3. Go to File - Export Audio.
  4. Select format: Opus file.
  5. Leave all other settings exactly as they are.
  6. Export the file.
  7. Rename the exported .opus file to .ogg.
  8. Done.
