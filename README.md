# Nothing Phone Glyph Composer

Web toolkit to create 13x13 glyph animations, prepare audio, and export ringtone-ready OGG files.

## Live Pages

- [Glyph Composer](https://itssveennn.github.io/Nothing-Glyph-Composer/index.html)
- [Audio Converter](https://itssveennn.github.io/Nothing-Glyph-Composer/audio-converter.html)
- [Gallery](https://itssveennn.github.io/Nothing-Glyph-Composer/gallery.html)
- [Tutorial](https://itssveennn.github.io/Nothing-Glyph-Composer/tutorial.html)

## What You Can Do

- Draw animations on a 13x13 matrix
- Adjust frame durations
- Sync animation timing to audio
- Convert audio to OGG Vorbis
- Edit audio with:
  - Trim start/end
  - Append silence
  - Volume adjustment
- Import/export animation JSON
- Save/load full projects with `.glyphproj`
- Bake glyph metadata into OGG files for ringtone use

## Quick Workflow

1. Open [Audio Converter](https://itssveennn.github.io/Nothing-Glyph-Composer/audio-converter.html), upload your audio, edit it, and convert to `.ogg`.
2. Use `Use in Glyph Composer` (or open the OGG manually in Composer).
3. Create and fine-tune your animation in [Glyph Composer](https://itssveennn.github.io/Nothing-Glyph-Composer/index.html).
4. Export with `Bake & Download OGG`.
